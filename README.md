# 📊 market-financial-dashboard
### Proyecto de análisis del mercado financiero utilizando Python.

---

## 📌 Descripción del Proyecto
Este proyecto analiza el comportamiento del mercado financiero mediante la extracción, limpieza y procesamiento de datos financieros reales. Incluye métricas clave como rendimientos, volatilidad, drawdown, Sharpe Ratio y comparativos entre distintos activos. Finalmente, se construye un dashboard interactivo en Power BI para visualizar patrones, riesgos y tendencias del mercado.

---

## 🎯 Objetivo
Crear un análisis completo e interactivo del mercado financiero usando Python, SQL y Power BI, demostrando habilidades en:

- Extracción de datos financieros (APIs/datasets)
- Procesamiento y transformación de datos
- Cálculo de indicadores financieros
- Visualización de insights
- Storytelling financiero

---

## 🛠️ Tecnologías utilizadas
- Python (Pandas, NumPy, yFinance, Matplotlib)
- SQL (consultas para manipulación y análisis)
- Power BI (Dashboard interactivo)
- GitHub (control de versiones)
- Excel / CSV (almacenamiento de datos)
